# 🤖 AI Code Editor Bot

Телеграм-бот для автоматической модификации кода с помощью AI

## 📋 Возможности

- 📝 Отправка файла с кодом
- 💬 Описание желаемых изменений
- 🤖 AI анализирует и модифицирует код
- 📂 Получение готового файла
- ⚙️ Переключение между AI моделями
- 👨‍💼 Админ панель

## 🚀 Установка

1. **Клонируйте репозиторий или скачайте файлы**

2. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

3. **Настройте .env файл:**
```env
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_ID=ваш_telegram_id
```

**Где взять данные для .env:**
- `BOT_TOKEN` - получите у [@BotFather](https://t.me/BotFather) создав нового бота
- `ADMIN_ID` - ваш Telegram ID, узнайте у [@userinfobot](https://t.me/userinfobot)

4. **Запустите бота:**
```bash
python bot.py
```

## 🎯 Как использовать

1. **Запустите бота** - `/start`
2. **Нажмите "📝 Изменить код"**
3. **Отправьте файл** с вашим кодом (.py, .js, .html и т.д.)
4. **Опишите изменения**, например:
   - "Добавь функцию для подсчета суммы"
   - "Измени переменную x на y"
   - "Удали функцию calculate"
5. **Получите готовый файл** с изменениями

## 📞 Поддержка

При возникновении проблем проверьте:
1. Правильность токена в .env
2. Доступность API
3. Наличие интернет-соединения

### 🔍 Диагностика проблем

**Если бот не отвечает на запросы AI:**
1. Используйте команду `/test` (для админа) чтобы проверить соединение с API
2. Проверьте логи бота в консоли - там будут детали ошибок
3. Попробуйте сменить модель AI через "⚙️ Сменить модель"
4. Убедитесь что у вас есть интернет соединение

**Если AI не может применить изменения:**
1. Формулируйте запросы конкретнее
2. Указывайте точные имена функций/переменных
3. Разбивайте сложные изменения на несколько простых запросов
4. Попробуйте другую AI модель (например, gemini-3-pro)

**Логирование:**
Бот записывает подробные логи всех операций в консоль:
- Запросы к API
- Ответы от AI (включая статус коды)
- Применение изменений
- Ошибки и предупреждения
