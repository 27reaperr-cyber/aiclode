# 🚀 Быстрый старт AI Code Editor Bot

## 📋 Шаг 1: Установка

```bash
# Установите зависимости
pip install -r requirements.txt
```

## 🔑 Шаг 2: Настройка .env

Откройте файл `.env` и заполните:

```env
BOT_TOKEN=получите_у_@BotFather
ADMIN_ID=ваш_telegram_id  # Узнайте у @userinfobot
```

**Где взять:**
- `BOT_TOKEN` - у [@BotFather](https://t.me/BotFather)
- `ADMIN_ID` - у [@userinfobot](https://t.me/userinfobot)

## ▶️ Шаг 3: Запуск

```bash
python bot.py
```

## 🎯 Первое использование

1. Откройте бота в Telegram
2. Отправьте `/start`
3. Проверьте соединение с API: `/test` (только для админа)
4. Нажмите "📝 Изменить код"
5. Отправьте файл с кодом
6. Опишите что нужно изменить

## ✅ Проверка работы

Если команда `/test` показывает "✅ API работает!" - всё настроено правильно!

## 🆘 Проблемы?

**Ошибка связи с AI:**
- Проверьте интернет
- Попробуйте команду `/test`
- Смените модель через "⚙️ Сменить модель"

**Бот не отвечает:**
- Проверьте BOT_TOKEN в .env
- Убедитесь что бот запущен (python bot.py)

**AI не может изменить код:**
- Формулируйте запросы конкретнее
- Указывайте точные имена функций
- Попробуйте другую модель (gemini-3-pro рекомендуется)
